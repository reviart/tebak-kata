# Cara bermain
1. Pemain akan memilih berapa banyak soal yang ingin dijawab
2. Setiap soal pemain dapat memilih banyaknya kesempatan mencoba
3. Pertama kali pemain memiliki level Beginner, dan level tertinggi adalah Hard

# Level
1. Beginner (point +1)
2. Medium (point +2)
3. Hard (point +3)
